# Cómo clonar e instalar el proyecto

Abre tu terminal o consola.

Clona el repositorio en tu máquina local ejecutando:

git clone https://github.com/PowerSystem2024/DynamicDevs_ProyectoIntegrador_CuartoSemestre_Front.git


Ingresa a la carpeta del proyecto:

cd DynamicDevs_ProyectoIntegrador_CuartoSemestre_Front

Instalaciones necesarias para poder correr el proyecto en tu pc

- npm install (Creara la carpeta node_modules) con todas sus dependencias.

npm install bootstrap
npm install react-bootstrap
npm install bootstrap-icons
npm install react-bootstrap-icons
npm install react-hook-form
npm install sweetalert
npm install sweetalert2
npm install json-server
npm install react-icons
